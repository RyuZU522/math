# MathViz Dashboard

数学概念与公式可视化看板 —— 基于 Vue 3 + Vite + TypeScript + KaTeX + Canvas。

## 功能

- KaTeX 渲染 LaTeX 公式
- 搜索与分类筛选（线性代数 / 微积分 / 微分方程）
- Canvas 交互可视化：Jordan 块稳定性、傅里叶级数、阻尼谐振子、矩阵变换、Lotka–Volterra、泰勒级数
- 暗色玻璃拟态 UI，参数滑块实时重绘

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
